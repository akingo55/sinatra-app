# sinatra-app
This is the memo app using by sinatra.

## setup
1. execute the following command:
```bash
bundle install
```

2. copy env file and set necessary environment variables
```bash
cp ~/.env.sample ~/.env
```

3. source environment variables
```bash
source ~/.env
```

## execution
1. run the script
```bash
ruby memo_app.rb
```
2. access to `http://localhost:4567/`
